## Campus Events Portal project – Serverless Web Application on AWS
# Architecture

Frontend → API Gateway → AWS Lambda → DynamoDB → SNS
Authentication handled by Amazon Cognito

# Main components:

Static Website hosted on Amazon S3

CDN delivery using Amazon CloudFront

REST APIs built with Amazon API Gateway

Backend business logic using AWS Lambda

Data storage using Amazon DynamoDB

Authentication using Amazon Cognito

Notifications using Amazon SNS

DNS routing using Amazon Route 53


| Service            | Purpose                                |
| ------------------ | -------------------------------------- |
| Amazon S3          | Static website hosting                 |
| Amazon CloudFront  | Global CDN for faster content delivery |
| Amazon API Gateway | REST API management                    |
| AWS Lambda         | Serverless backend logic               |
| Amazon DynamoDB    | NoSQL database for event data          |
| Amazon Cognito     | User authentication and authorization  |
| Amazon SNS         | Event notification system              |
| AWS IAM            | Role-based access control              |
| Amazon Route 53    | Domain management and DNS routing      |
